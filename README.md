## Setup

This project uses `Create React App` and [ReScript/ReasonML](https://rescript-lang.org).

### Install dependencies
```sh
yarn
```

### Development mode
- ReScript compiler:
```sh
yarn bsb
```

- CRA:
```sh
yarn start
```

### Build
```sh
yarn build
```
